📊 Project Management Portfolio
A comprehensive and professionally designed Excel-based project management portfolio with multiple tracking sheets, dynamic formulas, and attractive visualizations.

🌟 Features
Executive Dashboard - Real-time overview of all projects and budgets
Detailed Project Tracking - Complete project information with status monitoring
Timeline Management - Visual timeline with duration and deadline tracking
Budget Analysis - Financial tracking with utilization metrics
Dynamic Formulas - All calculations update automatically
Professional Design - Color-coded status indicators and clean formatting

🚀 How to Use
Download the `project_portfolio.xlsx` file
Open in Microsoft Excel, Google Sheets, or LibreOffice Calc
Navigate between sheets using the tabs at the bottom
Customize by replacing sample data with your actual projects
Monitor your portfolio using the Dashboard for quick insights
✏️ Customization
Adding New Projects
Go to the Projects sheet
Add a new row below the existing projects
Fill in: Project ID, Name, Owner, Status, Priority, Dates, Budget, Spent, and Progress
The Dashboard, Timeline, and Budget Tracking sheets will update automatically
Modifying Formulas
All formulas are designed to be dynamic. The sheets reference each other:
Dashboard pulls data from Projects sheet
Timeline extracts date information from Projects sheet
Budget Tracking calculates financial metrics from Projects sheet
